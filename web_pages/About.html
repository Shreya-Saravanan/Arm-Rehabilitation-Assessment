Abstract

  • Stroke is becoming a major cause of early mortality and disability in low- and middle-income countries such as India. Muscle atrophy occurs in hemiplegic individuals as a result of underused muscles, necessitating therapy in order to fully recover.
  • The arm rehabilitationÂ is performed by continuous repeating workouts intended at treating hemiplegic patients while providing feedback on their activities while executing the exercise set in terms of their recovery rate. The degree of recovery of the hemiplegic patient is determined by the ease and speed with which an exercise is done effectively.
  • Our goal is to provide a low-cost solution to a severe problem affecting the country's low- and middle-income groups.
  • We were able to accomplish this using a webcam-based system that uses Deep Learning Models to provide Rehabilitation Assessment to Eight Arm Rehabilitation Exercises that users can access via a Web Application.

About the Web Application

  • The Web Application was developed using Flask Framework coupled with basic HTML, CSS and JavaScript.
  • The Flask Application makes use of pre-trained Deep Learning Models for each Exercise.
